# Password Strength Checker using Python

## Project Description

This project is a simple Password Strength Checker developed using Python. The program checks the strength of a password based on its length and categorizes it as Weak, Medium, or Strong.

## Features

- User password input
- Password length validation
- Weak password detection
- Medium password detection
- Strong password detection
- Beginner-friendly project

## Technologies Used

- Python

## Concepts Practiced

- Variables
- Strings
- len() Function
- Conditional Statements (if, elif, else)
- Logic Building

## Password Rules

- Length less than 6 → Weak Password
- Length between 6 and 10 → Medium Password
- Length greater than 10 → Strong Password
